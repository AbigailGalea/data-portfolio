Overview

This project focuses on sentiment analysis of Maltese news articles published on Times of Malta. The goal was to understand public sentiment around key national events and track how it evolved over time.

Using web scraping, natural language processing, and time series analysis, I built an end-to-end pipeline to collect, process, and analyse over 20,000 articles. The results reveal interesting patterns in how sentiment shifts in response to political, social, and economic events.

Technologies Used
  • Python — Data collection, preprocessing, analysis
  • R — Time series analysis & visualisation
  • Selenium — Web scraping
  • SpaCy / NLTK — Text processing
  • Matplotlib & Plotly — Charts & dashboards
Key Insights
  • 🔻 Sentiment drops significantly during major political events — negative peaks correlate with elections and scandals.
  • ➖ Neutral sentiment dominates daily articles — nearly 46% are neutral, indicating factual reporting.
  • 🔺 Positive sentiment grows over time — a gradual increase visible in 2022.
